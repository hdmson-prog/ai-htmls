System Philosophy

Modular Electrical Logic

Our switch systems operate on a unified modular architecture. Each component—switch, dimmer, socket, or data interface—shares consistent mounting geometry and electrical specifications.

Interchangeability is engineered at the system level. One back box accommodates multiple module types. One installation method serves all configurations.

This is not product variety. This is systematic precision designed for architectural consistency across entire projects.

KNX Integration

Native KNX TP1 support. Every switch transmits and receives KNX telegrams. No gateway required for KNX installations.

  • Group objects: 254 per device
  • Communication: TP1 (twisted pair)
  • Power supply: KNX bus (30V DC)

DALI / DMX Control

Integrated DALI master for luminaire control. DMX output for architectural lighting sequences.

  • DALI channels: 64 per dimmer
  • DMX universes: 1 (512 channels)
  • Scenes: 16 programmable

BACnet / Modbus

Commercial building integration via BACnet IP or Modbus TCP. BMS-agnostic deployment.

  • BACnet/IP compliant (B-ASC)
  • Modbus TCP slave
  • REST API for monitoring

RF Wireless

868 MHz RF option for retrofit installations. Maintains full functionality without wired back box.

  • Range: 30m indoor
  • Battery: 5-year life
  • Encryption: AES-128

# 🚀 Quick Start Guide

## Instant Setup (3 Steps)

### 1️⃣ Open the Website
Double-click `index.html` or open it in your browser.

### 2️⃣ Customize Content
- **Logo & Brand Name**: Search for "Lumière" in `index.html` and replace
- **Colors**: Edit CSS variables at the top of `styles.css`
- **Images**: Replace Unsplash URLs with your product images

### 3️⃣ Deploy
- Upload files to your hosting or
- Use GitHub Pages / Netlify for free hosting

## 🎨 Quick Customization

### Change Brand Colors
Open `styles.css` and edit (around line 10):
```css
--color-primary: #c9a86a;      /* Your primary color */
--color-secondary: #2c2416;    /* Your secondary color */
```

### Update Images
Search for image URLs in `index.html`:
```html
src="https://images.unsplash.com/..."
```
Replace with your image paths:
```html
src="images/your-product.jpg"
```

### Edit Contact Info
Find the contact section (line ~950) and update:
- Address
- Phone number
- Email
- Social media links
